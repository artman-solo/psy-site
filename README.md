# Сайт-визитка психолога

Профессиональный лендинг с адаптивной версткой, каруселью отзывов и интеграцией EmailJS.

## Технологии
* HTML5 / CSS3 (Tailwind CSS)
* JavaScript (Vanilla JS)
* EmailJS API для формы обратной связи

## Как запустить локально
Просто откройте `index.html` в браузере или используйте расширение **Live Server** в VS Code.
